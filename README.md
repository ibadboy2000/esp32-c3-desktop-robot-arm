# ESP32-C3 桌面机械臂控制系统

基于 ESP32-C3 的桌面级机械臂/车控制程序，支持通过网页（手机/电脑）进行控制，并支持物理手柄（通过 HTML5 Gamepad API）直连浏览器进行极低延迟控制。

## 🌟 功能特性
- **网页可视化控制**：内置前端控制界面，无需安装任何 APP。
- **物理手柄支持**：支持任何连接到电脑/手机的蓝牙或有线手柄（如 Xbox / PS / 盖世小鸡等），摇杆精准控制，按键映射。
- **智能配网 (WiFiManager)**：无需修改代码中的账号密码，首次开机会自动开启热点，通过手机连接即可配置家里 WiFi。
- **超低延迟防抖**：针对摇杆的模拟量输出进行了节流优化，解决了网络请求风暴导致的卡顿。
- **硬件保护**：重新校准了 PWM 信号频率（20ms周期），防止舵机抖动和烧毁；开启了欠压保护。

## 🛠️ 1. 烧录与下载教程

### 准备工作
1. 下载并安装 [Visual Studio Code (VSCode)](https://code.visualstudio.com/)。
2. 在 VSCode 的扩展商店中搜索并安装 **PlatformIO IDE** 插件。
3. 准备一根数据线，将 ESP32-C3 开发板（WEMOS LOLIN C3 Mini）连接到电脑。

### 编译与烧录
1. 在 VSCode 中，点击 `文件 -> 打开文件夹`，选择本项目的根目录。
2. PlatformIO 会自动初始化工程并下载相关的依赖库（首次可能需要几分钟）。
3. 点击 VSCode 底部状态栏的 **“✓” (Build)** 按钮进行编译。
4. 编译成功后，点击底部状态栏的 **“→” (Upload)** 按钮进行烧录。
> **注意**：部分 ESP32-C3 开发板在烧录时，可能需要按住板子上的 `BOOT` 键，然后点击一次 `RST` 键进入下载模式。

## 🌐 2. 首次配网教程 (WiFiManager)

代码中**不需要**硬编码 WiFi 密码，系统采用了智能配网模式。

1. **上电启动**：烧录完成并拔插电源后，ESP32 启动。此时它发现没有连接过 WiFi，会自动开启一个热点。
2. **连接热点**：打开手机或电脑的 WiFi 列表，找到名为 `ESP32_Arm_Setup` 的无密码 WiFi 并连接。
3. **进入配网页面**：连接后，通常手机会自动弹出一个配网门户页面。如果没有自动弹出，请打开浏览器，访问 `192.168.4.1`。
4. **配置网络**：点击 `Configure WiFi`，在列表中选择您家里的 2.4G WiFi（注意：ESP32 不支持 5G WiFi），输入密码并保存。
5. **配网成功**：ESP32 会自动重启并连接到您家里的路由器。
> **重置网络**：如果您想让它连接到其他 WiFi，可以使用 PlatformIO 的 Erase Flash 功能清空主板，或者在代码中加入清除配网的逻辑。

## 🎮 3. 网页访问与手柄控制教程

### 访问控制页面
1. 确保您的电脑/手机与 ESP32-C3 连接在**同一个路由器的局域网**下。
2. 将 ESP32 通过串口连接到电脑，打开 PlatformIO 的 Serial Monitor（波特率 115200），获取板子被分配的局域网 IP 地址（例如 `http://192.168.31.x`）。
3. 在浏览器中输入该 IP 地址，即可打开控制界面。

### 手柄连接与控制
1. 将您的物理手柄（如 Xbox / PS / 盖世小鸡）通过蓝牙或接收器/数据线连接到 **您的手机或电脑**。
2. 保持刚刚打开的 ESP32 控制网页**在最前台**。
3. **唤醒手柄**：在网页打开的情况下，随便按一下手柄上的任意按键。网页顶部会提示 `✅ 游戏手柄已连接: 您的手柄型号`。
4. **开始操作**：
   - **左摇杆**：控制 TT 电机（底盘前后/上下）。
   - **右摇杆**：控制舵机 1 和 舵机 2。
   - **L1/L2 按键**：张开机械爪（舵机 3）。
   - **R1/R2 按键**：闭合机械爪（舵机 3）。
   - **B 键 / 圆圈键**：机械爪归中。

> ⚠️ **重要提示**：手柄信号是通过浏览器的 `Gamepad API` 获取并转发给 ESP32 的。因此，**控制期间必须保持浏览器处于打开并亮屏的状态**。如果浏览器被切到后台或手机息屏，系统会挂起网页，手柄控制将失效。
